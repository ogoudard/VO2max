# DIY 3L Calibration Pump

This project describes a simple 3-liter calibration pump used to calibrate airflow measurements (e.g. Venturi + differential pressure sensor systems).

## Overview

The pump is built using:
- PVC tube (Ø 100 mm) length > 3.0E6 */(pi * r^2)
- Pipe (Ø 8 mm) lenght match PVC tube
- 3D printed parts (piston and end caps)

## Design

### Piston Assembly
- `Piston.stl`
- `Nut.stl`
- `Ferrule.stl`

### End Caps
- `Vis.stl` (x2)
- `EndCap.stl`
- `Embout.stl`
- `Embout1.stl`

## Construction

The piston slides inside the PVC tube to displace a known volume of air (3 liters).  
The copper pipe is just a rod to push the piston
