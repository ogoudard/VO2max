VO2 MAX sensor based on https://www.instructables.com/Accurate-VO2-Max-for-Zwift-and-Strava/

This repository includes:
  - Arduino code (deprecated: changed for Embedded Software)
  - Embedded Software more advance software control
  - iOS App (deprecated: changed for Cross Platform Flutter App)
  - Flutter App: App runnable on Windows, Linux, MacOS, iOS and Android to visualize and save VO2max test measurement
  - CAD and printing files of mechanical enclosure and calibration pump
  - Python script to display and save in a CSV file all measured data through USB connection
